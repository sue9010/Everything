CSS

CSS 는 Cascading Style Sheets를 의미한다.

Cascading: 계단식
Style: 멋을 내다 Sheet: (종이) 한 장

==>계단식으로 스타일을 정의하는 문서!

CSS 문서(코드) 자체는 독립적으로 존재할 수 있지만,
그 목적이 구조화된 HTML 문서의 스타일을 정의하는데 있으므로
HTML문서가 없는 CSS 문서는 사실상 의미가 없다.

HTML: 웹 문서의 콘텐츠를 구성하는 언어
CSS: 웹 문서의 콘텐츠에 스타일을 추가하는 언어 (색상, 크기, 위치 등)

=>HTML과 함께 사용되지 않는 CSS는 단순 텍스트일 뿐이며, 웹 브라우저를 통해 결과물을 확인하기 위해서는 반드시 HTML 문서 작성이 선행되어야 함
